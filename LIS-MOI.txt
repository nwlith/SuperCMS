Hello !

Plusieurs choses avant d'utiliser cela :

- Git c'est super bien, tu devrais transformer ce projet en dépôt git
tu remarqueras qu'il y a un nouveau fichier qui s'appelle ".gitignore"
il contient les choses que git doit ignorer : on ne met jamais la base
de données dans git, et les modules node non plus puisqu'ils s'installent
tous seuls

pour ce faire: sudo apt install git
puis, dans ce dossier:
git init (crée un dépôt local)
git add . (ajoute tous les fichiers)
git commit -m "Mon premier commit depuis un terminal"

là, il te mettra un message pas content à propos de ton nom et ton mail
pour les commits : il suffit de le lire attentivement pour régler cela :)

- Ensuite, tu dois donc créer un fichier db.sqlite comme sur ton ordi puisque je l'ai exclu de ce dossier

- Enfin, npm i et tout sera disponible :)

cheers \o/
